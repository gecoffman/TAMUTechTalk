 
*** Requirements

python 
pandas


*** Getting started 

- Equities
Run the equtites server located in the equtites directory (see ReadMe.txt
in that directory for instructions)

While the service is running, run the solution. 

>> python equtitiesScraper.py


- Forex
Run the forext server located in the forex directory (see ReadMe.txt in that
directory for instructions) 

While the service is running, run the solution with the following command. 

>> python forexScraper.py 

*** Output

When the above solutions are run, they should populate the directory with CSV files for the Equities data and JSON files for the Forext data.
