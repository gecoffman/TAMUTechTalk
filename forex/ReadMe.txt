

*** Requirements 

python
flask 



*** Getting started 
In your terminal, navigate to the service.py file. 

Type the follwoing command to run the server

>>> python service.py

In a browser, go to http:127.0.0.1/8080/. 

You should see a web page showing the available currencies. 

To view the raw JSON data, add '/forex/A_CURRENCY_PAIR/' to the end of the URL. 

This would make the whole URLhttp:/127.0.0.1/8080/forex/USDJPY if you wanted to view the USDJPY data. 

While the service is running, you can run the given solution. 

Open a second terminal and follow the instructions for the solution to simulate a server broadcasting data and a client connecting and downloading the data.
